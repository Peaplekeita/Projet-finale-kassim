Projet 1 : [Calculateur de prêt hypothécaire]

Objectif

Concevoir une application web simple permettant de calculer automatiquement le paiement mensuel d'un prêt hypothécaire à partir de données saisies par l'utilisateur.

Fonctionnalités développées :

Interface HTML avec champs de saisie pour le montant du prêt, le taux d'intérêt annuel et la durée du prêt.

Script JavaScript pour effectuer le calcul du paiement mensuel à l'aide de la formule mathématique standard.

Validation des données saisies (valeurs positives, formats valides).

Gestion des erreurs utilisateur via des boîtes d'alerte conviviales.

Tests fonctionnels avec différents scénarios d'entrée.

Compétences acquises

Intégration HTML/CSS/JavaScript dans un projet fonctionnel.

Manipulation du DOM, gestion des événements et calculs dynamiques en JavaScript.

Validation des formulaires et gestion des erreurs.

Amélioration de l'expérience utilisateur avec une interface simple et intuitive.

Outils utilisés :

Visual Studio Code pour l'écriture et l'organisation du code.

HTML5, CSS3, JavaScript pour le développement.

Navigateurs web pour les tests (Chrome, Firefox, etc.)

Projet : Manipulation de dates avec JavaScript

Objectif :

Créer une interface utilisateur permettant de saisir une date, d'en extraire ses composants (jour, mois, année) et de calculer la différence en jours avec la date actuelle.

Fonctionnalités développées :

Formulaire HTML permettant la saisie d'une date au format "JJ-MM-AAAA"

Conversion de la date saisie en objet Date JavaScript

Calcul en temps réel de la différence de jours entre la date saisie et la date du jour

Compétences acquises :

Maîtrise de la manipulation du DOM pour récupérer et afficher dynamiquement des données

Utilisation approfondie du constructeur Date() de JavaScript

Gestion du format de date personnalisé (français)

Mise en œuvre de calculs temporels (différence en millisecondes, conversion en jours)

Débogage et validation côté client pour garantir une expérience utilisateur fluide

Outils utilisés :

HTML5, CSS, JavaScript pur, navigateur Web (affichage interactif du résultat)
