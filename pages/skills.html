Technologies maîtrisées

Front-end

Compétences en HTML5

Bonne maîtrise du langage HTML5, avec une compréhension approfondie du balisage et de la sémantique des balises. Capacité à structurer des pages web en utilisant des balises adaptées, à intégrer des liens hypertextes, des images, des tableaux et des formulaires. À l'aise avec l'ajout d'éléments de formulaire (champs de saisie, boutons, etc.) en fonction des types de données, conformément aux standards du web

Compétences en CSS

Bonne maîtrise des feuilles de style CSS pour la mise en forme de pages web esthétiques et cohérentes. Capacité à utiliser efficacement les différents types de sélecteurs (classe, ID, élément) pour styliser précisément les balises HTML et assurer une présentation harmonieuse du contenu.

Compétences en JavaScript

Capacité à intégrer du JavaScript dans des pages HTML pour rendre les interfaces interactives. Maîtrise des variables, opérateurs, structures conditionnelles, boucles et tableaux. Compétent dans l'écriture de fonctions modulaires, ainsi que dans l'utilisation des objets standards (Math, String, Date, Array) pour effectuer des opérations courantes et manipuler efficacement les données.

Projet Portfolio Web

Réalisation d'un site web personnel mettant en œuvre les compétences acquises en HTML5, CSS3 et JavaScript. Conception de pages HTML bien structurées, application de styles CSS cohérents (palette de couleurs, typographie) et ajout de fonctionnalités interactives via JavaScript (validation de formulaires, effets visuels). Ce projet concret a permis de renforcer ma compréhension du développement web et de me confronter à un flux de travail réaliste.

Utilisation optimale de l'éditeur Visual Studio Code pour le développement web : gestion efficace des extensions, navigation fluide dans les fichiers, utilisation des fonctionnalités avancées ( terminal intégré, coloration syntaxique, auto-complétion, débogage) pour accélérer l'écriture et la maintenance du code HTML, CSS et JavaScript.
